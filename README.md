# ThemeAutumn

## Description

A warm and muted autumn theme for Matomo with burnt oranges, deep burgundy, and golden harvest accents. Evokes falling leaves.

This theme transforms your Matomo dashboard with a rich, earthy color palette featuring:

- Warm cream and beige backgrounds with deep earth text tones
- Deep burgundy header with golden accents
- Custom graph color series in burnt oranges, amber golds, cranberry, and forest greens
- Visitor map styled with amber gradients and golden highlights

![ThemeAutumn Dashboard](screenshots/Autumn_Dashboard.png)

## Installation

Install it via Matomo Marketplace or manually:

1. Download the plugin
2. Extract the files into the `plugins/ThemeAutumn` directory
3. Activate the theme in **Administration > Platform > Themes**
